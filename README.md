# Python-Project
Desenvolver uma aplicação em Python, estruturada, que permita a gestão de utilizadores e passwords.  A aplicação deverá permitir o registo, pesquisa, edição, remoção de utilizadores, usando estruturas de  dados adequadas.
Os dados principais dos utilizadores são nome, morada, contacto e password.
